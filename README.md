# pulsepoint-news

A simple Firebase Hosting landing page for the PulsePoint News project.

## Deploy

To deploy with a Firebase CI token, set `FIREBASE_TOKEN` and run:

```bash
cd /workspaces/pulsepoint-news
FIREBASE_TOKEN=your_token_here npx firebase deploy --only hosting --project pulsepoint-news
```

If you want me to continue, please paste the `FIREBASE_TOKEN` here.
